La buena práctica para la programación:
- La programación ayuda mucho en lo que es el mantener una consiente o constante tabulación en los códigos.
- Al momento de comentar tu código, se debe evitar comentarios obvios de este, es decir, ser redundante.
- Nos hace entender que las lineas de código pueden ser 
separadas llamados bloques, dando a entender que la agrupación de códigos es muy beneficioso.
- La organización de los códigos en carpetas o archivos.
- Mantenimiento de un buen control sobre todas las versiones del código.
